# Shanmengshe

校园户外社团网站项目，用于成员管理、活动记录、装备管理和知识分享。

## 技术栈

- **前端：** React + Next.js（TypeScript）用于组件化开发与 SSR，提升性能与 SEO。
- **样式：** Tailwind CSS，快速实现定制化界面。
- **后端：** NestJS（Node.js + TypeScript），模块化架构，适合构建可维护的 API。
- **数据库：** PostgreSQL，提供强一致性和可靠的事务能力，用于用户、装备等核心数据。
- **部署：** Docker，将应用环境容器化，保证开发、测试、生产一致性，简化发布流程。
